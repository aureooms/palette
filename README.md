# Palette

> :art: A color palette compositor.

A [live demo](https://aureooms.github.io/palette) is running on github pages.

## Develop

```js
yarn install
```

```js
yarn start
```

> :warning: We are using parcel v2. Currently alpha/beta stage. Crashes often.

To remove parcel's cache

```js
rm -r .parcel-cache
```


## Build

```js
yarn build
```
